# moyu

摸鱼赚钱

```
我想写一个桌面小程序供人使用。功能大概是在上班期间，能够看到自己赚取的工资实时增长；初始化自己的月工资、上班时间等信息后，程序根据时间计算每天已挣了多少钱，当月挣了多少钱。
请根据上述内容，帮我生成一个完整的项目代码及部署方案，使用VS2022开发WPF应用程序，使用.Net Framework3.5，能够在windows7系统上运行，要求界面简洁白底黑字，小窗口无标题栏，可拖到角落，可置顶运行，配置数据本地保存。需要显示当前日期时间、上班状态（正在上班中、下班、等待上班）、今日收入、今日工作进度（进度条）、今日工作时长、本月收入、时薪、本月工作日，时间实时更新，工资数据3秒更新一次。

页面1：
设置你的工资信息开始使用
月工资（元）：[输入工资]
上班时间：[选择时间，默认09:00 ]
下班时间：[选择时间，默认18:00 ]
{按钮：开始赚钱}

页面2：
2025/08/13 16:00:32
正在上班中/已下班/等待上班 （用不同颜色显示）
今日已赚取 ￥399.30
今日工作进度(进度条展示)
{ 3.5 小时 \n 今日工作时长} { ￥34 \n 时薪 }
{ 12/21 \n 本月工作天数} { ￥3999.91 \n 本月收入}

其他说明：右键页面2菜单可选择“重新设置”、“置顶或取消置顶”、“退出”。请注意一定要使用.Net Framework3.5的语法。
```
